Updates one or many documents
Parameters:
Name Type Argument Default Description
selection Object | Array | String <optional>
 {} The selection for matching documents
update Object <optional>
 {} The update operation
options Object <optional>
 Additional options
Properties
Name Type Argument Default Description
updateAsMongo Number <optional>
 true By default: If the [update] object contains update operator modifiers, such as those using the "$set" modifier, then:
  • The [update] object must contain only update operator expressions
  • The Collection#update method updates only the corresponding fields in the document
      • If the [update] object contains only "field: value" expressions, then:
      • The Collection#update method replaces the matching document with the [update] object. The Collection#update method does not replace the "_id" value
      • Collection#update cannot update multiple documents
    override Number <optional>
    		 false Replaces the whole document (only apllies when [updateAsMongo=false])
    upsert Number <optional>
    		 false Creates a new document when no document matches the query criteria
    multi Number <optional>
    		 false Updates multiple documents that meet the criteria
    writeConcern Object <optional>
    		 null An object expressing the write concern
    callback function <optional>
    		 null Callback function to be called at the end with the results
    Returns:
